# Microsoft FastTrack Open Source - Set-ForwardingSMTPAddress

This script will set forwarding addresses for users specifed in a CSV file

## Usage

1. Copy the script file "Set-ForwardingSMTPAddress.ps1" and the users.csv template to a folder and open a PowerShell command window to that folder
2. Update the users.csv file to include the information for each user whose forwarding address you wish to set.
3. Execute the script

`.\Set-ForwardingSMTPAddress.ps1`

4. Use the dialog windows to login to your account, select the users.csv file to process, and select the report export location.


### All Options

This script takes no options

## Applies To

Exchange Online
